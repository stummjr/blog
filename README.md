# blog
My personal blog


## How to deploy

1. Commit the changes to this repo.
2. Run: 
```
$ ./bin/deploy.sh
```

## How to edit the theme

1. Run the theme project:
```
$ cd themes/hugo-lamp
$ yarn install
$ yarn run watch
```
2. Edit the theme as you wish. 
3. Run the hugo server.
```
$ hugo server -D
```

You can view the preview in http://localhost:1313
